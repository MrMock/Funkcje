# Funkcje

Program generujący wykresy funkcji sin(t), gdzie t jest czasem w sekundach, oraz szumu o rozkładzie Gaussa.

Do napisania użyłem bibliotek:
SFML ( https://www.sfml-dev.org/index.php ) - graficzne przedstawienie programu ( funkcje, skala, przyciski...),
time.h - pobieranie aktualnego czasu,
cmath - biblioteka pomocnicza przy obliczaniu.

dodatkowo do prawidłowego działania programu niezbędne jest posiadanie dowolnej czcionki i umieszczenie jej w katalogu z projektem z nazwą "arial.ttf"

całość wykonana w 3 dni po około 1-2h wliczając w to projekt, jak i pisanie programu.

link do prezentacji https://www.youtube.com/watch?v=KWRnXyWQIjo
